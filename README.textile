h1. SCALATEST EMBED MONGO

This project provides API to use an "embedMongo database":https://github.com/michaelmosmann/embedmongo.flapdoodle.de in your tests

It is directly inspired by "spec2-embedmongo":https://github.com/athieriot/specs2-embedmongo project.

This works in a Java 6+ environment

h2. Installation

h3. Git repo

scalatest-embedmongo is a SBT project.
It use 0.12 sbt version.

h3. Maven dependency

<pre><code>
    <dependency>
        <groupId>com.github.simplyscala</groupId>
        <artifactId>simplyscala-server_2.9.2</artifactId>
        <version>0.3</version>
    </dependency>
</code></pre>

h3. SBT dependency

<pre><code>
    libraryDependencies += "com.github.simplyscala" %% "simplyscala-server" % "0.3"
</code></pre>

    Only the following Scala versions 2.9.0, 2.9.1 & 2.9.2 are currently supported.

h4. Play! specific dependency management

In your play! project Build file :

<pre><code>
    object ApplicationBuild extends Build {

        val appDependencies = Seq (
            "com.github.simplyscala" %% "simplyscala-server" % "0.3"
        )
    }
</code></pre>

h3. Simple JAR file

"0.3 jar version":https://oss.sonatype.org/content/repositories/releases/com/github/simplyscala/simplyscala-server_2.9.2/0.3/simplyscala-server_2.9.2-0.3.jar

h2. Usage

In order to use SimplyScala server API, import as follows:

<code>import fr.simply._</code>

h3. Basic Usage

<pre>
<code>
    val server = new StubServer(8080).start
    server.stop
</code>
</pre>

By default, with no routes defined, SimplyScala server returns a 404 text/plain response with "error" in body
